# Recode

Recode is a fresh Tauri-based AI code editor. It starts with a CodeEdit-inspired desktop shell, an Athas-style hybrid custom editor strategy, and a Rust backend shaped for filesystem, terminal, git, LSP, and AI agent features.

## Development

```bash
pnpm install
pnpm dev
```

The first implementation slice includes the app shell, navigator, tabs, textarea-backed custom editor surface, terminal placeholder, AI inspector placeholder, command palette, and Rust command scaffolding.
